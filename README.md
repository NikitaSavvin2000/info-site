
# Streamlit Tool


#### Установка зависимостей
```bash
pip install pdm
pdm install
```


Активация окружения
```bash
source .venv/bin/activate
```


Запуск на своей машине
```bash
python -m src.server
```


## Запуск через Docker

```bash
docker build -t streamlit_tool .
```
